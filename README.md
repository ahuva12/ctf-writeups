# CTF Writeups

This repository contains my solutions and writeups for various **Capture The Flag (CTF)** challenges.

Recently I began exploring the field of **cybersecurity**, with a particular interest in **low-level security**, including binary exploitation, memory vulnerabilities, and reverse engineering.

As part of this learning process, I began solving CTF challenges and documenting the full solving process.

The goal of this repository is to:

* Document my learning journey in cybersecurity
* Practice analyzing vulnerabilities and exploitation techniques
* Share clear and structured writeups of the challenges I solve

Each writeup describes the thought process, analysis steps, and the final solution used to solve the challenge.

---

## Repository Structure

Writeups are organized by challenge category, such as:

-Binary Exploitation
-Reverse Engineering

Each challenge directory typically includes:

writeup.md – a detailed explanation of the solution

screenshots/ – relevant screenshots from the analysis process

More challenges and categories will be added as I continue learning and solving new CTFs.
